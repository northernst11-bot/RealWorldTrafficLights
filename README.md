# Real World Traffic Lights

Real World Traffic Lights is a Cities: Skylines II code mod that mirrors traffic light placement data so generated signal heads face the through road from the far side of the intersection, closer to real-world mast-arm signal layouts.

Inspired by [Dava787](https://www.twitch.tv/dava787).

## Features

- Moves generated traffic light visuals toward real-world far-side placement
- Mirrors signal placement and rotation data used by road prefab compositions
- Requests a redraw of existing road nodes and edges when a city loads
- Visual-only behavior: no traffic simulation, lane routing, signal timing, or pathfinding changes
